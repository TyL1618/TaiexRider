# TaiexRider 封測意見彙整

> 版本：v0.11.x　封測期：2026-06  
> 來源：首批封閉測試用戶（Android TWA）

## 原始意見（逐條）

| # | 意見原文 | 分類 |
|---|---------|------|
| 1 | 正規化太嚴重導致賽道不夠刺激，過於平緩 | 遊戲性 / 地形 |
| 2 | 希望車子的造型種類多一些 | 內容 / 自訂 |
| 3 | 每日排行榜地圖的選取邏輯？遇到漲停板/跌停板賽道會極短且單一無趣不崎嶇 | 遊戲性 / 地圖選取 |
| 4 | 希望不只有後翻，也能新增煞車鈕前翻滾 | 操控 / 技巧多樣性 |
| 5 | 整體運行流暢，但玩久容易單一無趣 | 遊戲性 / 長期留存 |
| 6 | 可增加購買系統，包括車子造型甚至車子特殊功能 | 商業模式 / 變現 |

## 分類分析

### 🎮 遊戲性與地形（#1 #3 #5）

**#1 正規化過度 → 賽道平緩**  
- 根因：`terrain.ts` 的 `heightRange` 與斜率夾平（`refPct`）設計目的是「讓所有股票都可玩」，但代價是抹平了波動大的股票原有的劇烈感。  
- 可能改法（未決策）：
  - 依原始價格波動率（σ）動態調整 `heightRange`，波動大的股票給更大的高度範圍。
  - 提供難度選項（普通 / 困難），困難模式降低或關閉斜率夾平。
  - 正規化基準改「前日收盤」而非開盤，放大日內振幅視覺落差。

**#3 每日排行榜地圖選取邏輯**  
- 現況：`fetchDailyMap.ts` 抓全台上市股（~1090 支）存入 `daily_map`；排名賽地圖選取目前以「難度最高者」（坡度總和）為標準。  
- 問題：純漲停或純跌停的股票（整天同方向，幾乎無折返）坡度「高」但地形單調——直上或直下、不崎嶇。  
- 可能改法（未決策）：
  - 選取標準改為「波動複雜度」（折返次數、峰谷數量、振幅標準差）。
  - 排除條件：漲停/跌停次數過多、或當日資料點過少（<50 點）的股票。
  - 每日候選 → 人工策展白名單輔助篩選。

**#5 玩久單一無趣**  
- 現況：核心操控只有「單指後翻」一種技巧，長期重複感強。  
- 關聯 #4（前翻）、#2/#6（車型/功能多樣性），見下方。

---

### 🏍️ 操控多樣性（#4）

**#4 增加前翻（煞車鈕）**  
- 技術可行性：高（現有 `airSpinMax` / `airSpinAccel` 體系，反向套用即可；需新增第二個觸控區域）。  
- 設計考量：
  - 前翻與後翻需要不同的計分、動畫方向。
  - 雙鍵操控增加學習曲線，需 UI 提示（前翻區 / 後翻區）。
  - 可先做成「選配」或「解鎖後功能」，不影響新手體驗。

---

### 🎨 車型與自訂（#2 #6）

**#2 車子造型多樣化**  
- 現況：單一車型（霓虹重機外觀，`public/bike.png`）。  
- 可能方向：多套 sprite 切換（不改物理體積，只換貼圖）；代碼量小、視覺差異大。

**#6 購買系統（車型 + 特殊功能）**  
- 這是商業模式建議，與原規劃（Phase IAP）方向一致：
  - 車型 cosmetic → 純外觀，IAP 或廣告解鎖
  - 車子特殊功能（例：更快的翻滾加速、特殊得分倍率）→ 需平衡設計，避免 Pay-to-Win 破壞排行榜公平性
  - 建議切分：外觀類 IAP 無爭議；功能類限制在「非排名賽模式」生效

## 優先級建議（供後續規劃參考）

| 優先 | 項目 | 工時估計 | 備註 |
|------|------|---------|------|
| 高 | #3 改善每日地圖選取邏輯（波動複雜度篩選） | 小（腳本側改 SQL 篩選條件） | 直接影響核心留存體驗 |
| 高 | #1 動態地形高度（依股票波動率調整） | 中（terrain.ts + fetchDailyMap） | 解決「平緩」根本問題 |
| 中 | #4 增加前翻操控 | 中（GameCanvas + UI） | 豐富技巧深度 |
| 中 | #2 多車型 sprite | 小～中（美術為主） | 視覺差異大、開發量小 |
| 低 | #6 IAP 購買系統 | 大（後端 + Google Play Billing） | Phase 4+ 規劃，需後端 |
| 低 | #5 長期留存（系統性） | 大 | 依賴上方多項改善合力解決 |

## 可用於回答官方問題的標準說法

**Q：你們如何處理用戶反饋？**  
A：透過封閉測試階段蒐集用戶意見，涵蓋遊戲性、操控、內容多樣性與商業模式等面向。反饋已分類並整合進開發路線圖，依影響範圍與實作成本排定優先順序。

**Q：用戶對遊戲的主要正面評價？**  
A：整體運行流暢（#5 前半段）；核心操控（單指翻滾）直覺易上手。

**Q：用戶提出哪些改善方向？**  
A：① 地形刺激度不足 ② 操控技巧多樣性 ③ 每日地圖選取邏輯 ④ 車型與外觀自訂 ⑤ 長期留存與變現系統。
